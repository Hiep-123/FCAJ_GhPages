### Kết quả đạt được tuần 7:

- Hiểu kiến trúc Data Lake trên AWS:
  - Thu thập dữ liệu bằng Kinesis Firehose
  - Lưu trữ dữ liệu trên Amazon S3
  - Xây dựng Data Catalog với AWS Glue

- Thực hành xử lý và chuyển đổi dữ liệu:
  - AWS Glue ETL
  - AWS Glue Studio
  - AWS Glue DataBrew
  - Chuyển đổi dữ liệu sang định dạng Parquet

- Phân tích dữ liệu với Amazon Athena:
  - Truy vấn dữ liệu trực tiếp trên S3
  - Tối ưu hiệu năng bằng Partitioning
  - So sánh Columnar Storage và Row-based Storage

- Xây dựng Dashboard với Amazon QuickSight:
  - KPI Dashboard
  - Line Chart
  - Pie Chart
  - Pivot Table
  - Interactive Dashboard

- Làm việc với Amazon DynamoDB:
  - Thiết kế bảng dữ liệu NoSQL
  - CRUD Operations
  - Query và Scan
  - Global Secondary Index (GSI)
  - DynamoDB Streams

- Tìm hiểu các mô hình nâng cao của DynamoDB:
  - Capacity Units & Partitioning
  - Sparse Index
  - Composite Key
  - Adjacency List
  - Write Sharding
  - Global Tables

- Thực hành Change Data Capture (CDC):
  - DynamoDB Streams
  - Kinesis Data Streams
  - AWS Lambda Integration
  - Dead Letter Queue (DLQ)

- Xây dựng kiến trúc Serverless Event-Driven:
  - DynamoDB
  - Lambda
  - Event Processing Pipeline

- Hiểu quy trình phân tích dữ liệu hoàn chỉnh:
  - Ingest
  - Store
  - Catalog
  - Transform
  - Analyze
  - Visualize
