## Lab 05 – Amazon Relational Database Service (Amazon RDS)

### Tasks:

- Create VPC and Security Groups
- Create DB Subnet Group
- Deploy EC2 Instance
- Create Amazon RDS Database
- Connect application to RDS
- Perform Backup and Restore

### Learned:

- Amazon RDS architecture
- Application-to-database connectivity
- Backup and restore mechanisms
- Database networking and security

---

## Lab 43 – Database Schema Conversion and Migration

### Tasks:

- Configure MSSQL and Oracle source databases
- Use AWS Schema Conversion Tool (SCT)
- Convert schemas:
  - MSSQL → Aurora MySQL
  - Oracle → MySQL
- Create Migration Projects
- Configure Migration Tasks and Endpoints
- Validate migrated data in S3
- Create Serverless Migration
- Configure Event Notifications
- Monitor Logs and Troubleshoot issues

### Learned:

- AWS database migration workflow
- Cross-platform schema conversion
- AWS Database Migration Service (DMS)
- Monitoring and troubleshooting migration tasks
