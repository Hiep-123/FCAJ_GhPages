### Kết quả đạt được tuần 1:

- Hiểu tổng quan về Cloud Computing và AWS:
  - Nắm được các mô hình Cloud: IaaS, PaaS, SaaS
  - Hiểu sự khác biệt giữa On-premise và Cloud

- Nắm được các nhóm dịch vụ chính của AWS:
  - Compute (EC2)
  - Storage (S3, EBS)
  - Networking (VPC)
  - Database (RDS)

- Thiết lập thành công môi trường AWS:
  - Tạo AWS Free Tier account
  - Cấu hình IAM (User, Group, Policy)
  - Kích hoạt MFA để bảo mật

- Cài đặt và sử dụng AWS CLI:
  - Cấu hình Access Key, Secret Key, Region
  - Thực hiện các lệnh cơ bản:
    - Kiểm tra cấu hình
    - Liệt kê region
    - Làm việc với EC2

- Thực hành triển khai EC2:
  - Tạo instance
  - Kết nối SSH
  - Gắn EBS volume

- Hiểu cách quản lý chi phí:
  - Tạo AWS Budgets
  - Theo dõi usage để tránh vượt Free Tier

- Có khả năng kết hợp:
  - AWS Console (giao diện web)
  - AWS CLI (command line)
  → để quản lý tài nguyên hiệu quả
