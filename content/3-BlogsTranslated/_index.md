### [Blog 1 - Dual-token Authentication for Nakama Game Servers with Amazon Cognito on AWS](3.1-Blog1/)
This blog explains how AWS combines Amazon Cognito and Nakama to create a secure and seamless authentication experience for multiplayer game systems. It introduces the dual-token model and shows how this architecture improves both security and user experience.

### [Blog 2 - AWS GenAIIC Partner Agent Factory – Bringing AI Agents to AWS Marketplace](3.2-Blog2/)
This blog introduces the Partner Agent Factory program from AWS, which helps organizations move AI agents from prototype stage to production-ready deployment. It highlights the value of Amazon Bedrock, AgentCore, and Strands Agents in accelerating enterprise AI adoption.

### [Blog 3 - Strengthening AWS Management Console Security with Sign-in Resource-based Policies and RCPs](3.3-Blog3/)
This blog discusses a new AWS security feature that restricts sign-in access to trusted networks using Sign-in Resource-based Policies and Resource Control Policies. It emphasizes the benefits of centralized policy management, auditability, and stronger access control.
